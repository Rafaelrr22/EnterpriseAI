# EnterpriseAI



> EnterpriseAI is a self-hosted enterprise AI platform designed to help organizations search, understand and interact with internal knowledge using Retrieval-Augmented Generation (RAG) and local Large Language Models.



\## 📖 Overview



EnterpriseAI is an enterprise knowledge management platform designed to help organizations search, organize and interact with internal documentation using local AI models.



The project aims to provide a secure, self-hosted AI assistant capable of understanding company documentation and answering questions based on trusted sources.
